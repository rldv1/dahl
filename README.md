# Dahl
Здесь будут разборы, исследования и различные PoC.

## poc.py
Проверяет, использует ли пользователь мессенджер **Telega (Dahl)**

```bash
python poc.py user_id
# {"ok": true, "dahl_status": 2}
```

`dahl_status`: `0` = никогда не использовал, `1` = нет активной сесии, `2` = есть активная сессия
